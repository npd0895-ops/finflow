Đồng bộ & Backup
Kết nối Google Sheets để dùng trên mọi thiết bị
☁️ Google Sheets — Auto Sync
Dữ liệu tự động đồng bộ giữa tất cả thiết bị của bạn qua Google Sheets.
⚡ Tự động đồng bộ
Push lên Sheets sau mỗi lần lưu · Pull về khi mở app
Chưa cấu hình
📋 Hướng dẫn thiết lập
Làm 1 lần duy nhất • Không cần biết code
1 Tạo Google Sheets
Vào sheets.google.com → Bấm + tạo bảng tính mới → Đặt tên tùy ý (VD: FinFlow Database)
→ Click chuột phải vào tab Sheet1 ở dưới → Rename → Đổi thành data (viết thường)
2 Mở trình chỉnh sửa Script
Trong Google Sheets → menu Extensions (Tiện ích mở rộng) → Apps Script
→ Một tab mới mở ra → Xóa toàn bộ code mặc định trong editor → Bấm nút bên dưới để copy code mới
3 Dán code và lưu
Dán code đã copy vào editor → Bấm 💾 Save (Ctrl+S) → Bấm ▶ Run → Chọn hàm testScript → Bấm Run
→ Nếu log hiện "✅ Script hoạt động tốt!" là thành công
4 Triển khai Web App
Trong Apps Script → bấm Deploy (góc trên phải) → New deployment
→ Bấm ⚙️ cạnh "Select type" → chọn Web app
→ Execute as: chọn Me  ·  Who has access: chọn Anyone
→ Bấm Deploy → Authorize access → Cho phép quyền truy cập
→ Copy Web app URL (dạng https://script.google.com/macros/s/.../exec)
5 Kết nối vào FinFlow
Dán URL vào ô "Web App URL" ở trên → Bấm 🔗 Kiểm tra kết nối
→ Đèn chuyển xanh là thành công → Bật công tắc ⚡ Tự động đồng bộ → Xong! 🎉
Từ giờ mọi thay đổi sẽ tự đồng bộ giữa tất cả thiết bị
✅ Đã copy! Mở Apps Script và dán vào (Ctrl+V)
⚠️ Lưu ý: Mỗi lần cập nhật code Apps Script, bạn cần Deploy lại (New deployment) và cập nhật URL mới vào app.
📱 Backup thủ công
💡 Gợi ý: Xuất JSON mỗi tháng và lưu vào Google Drive. Khi đổi máy, chỉ cần nhập file JSON lại là có toàn bộ dữ liệu.
